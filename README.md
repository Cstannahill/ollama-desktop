# Ollama Desktop Chat

Simple Tauri + React client that streams responses from a local Ollama daemon.

## Usage

1. Install dependencies

   ```bash
   pnpm install
   ```

2. Start the Ollama daemon

   ```bash
   ollama serve
   ```

3. Run the app

   ```bash
   pnpm tauri dev
   ```
